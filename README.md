# Vehicle_recognition
# Win(Power Shell)
### install ffmp if not already
  winget install ffmpeg
### Install library
  pip install -r requirements
### Train model
  ./run.bat
### Streamlit
streamlit run app.py

# Ubuntu
### install ffmp if not already
  sudo apt update
  sudo apt install ffmpeg
### Install library
  pip install -r requirements
### Train model
  ./run.sh
### Streamlit
  streamlit run app.py
